## Building Executable

``` $ cd server
    $ mkdir build
    $ cmake -S . -B build
    $ cmake --build build --target Multithread-TCP-Server
    $ cd .. && cd client
    $ mkdir build
    $ cmake -S . -B build
    $ cmake --build build --target TCP-Client
```
